# 🇬🇧 English

## Overview

Board & Card Games Suite is a desktop application developed with Python and PyQt6 that combines two popular games in a single interface:

- Chess
- UNO

The project includes multilingual support, modern themes, AI gameplay for Chess, and a clean graphical interface.

## Features

### Chess

- Complete chess board
- Legal move validation
- Check and checkmate detection
- Stalemate detection
- Pawn promotion
- Castling support
- En passant support
- Human vs Human mode
- Human vs AI mode

### UNO

- Full UNO card system
- Wild cards and Draw 4 cards
- Skip, Reverse, and Draw 2 actions
- Multiple AI opponents
- Automatic color selection logic for AI
- Win detection

### User Interface

- Light Theme
- Dark Theme
- English Language
- Persian Language
- Chinese Language
- Responsive GUI built with PyQt6

# 🇨🇳 中文

## 项目介绍

Board & Card Games Suite 是一个基于 Python 和 PyQt6 开发的桌面游戏应用程序，集成了两款经典游戏：

- 国际象棋（Chess）
- UNO 纸牌游戏

项目提供现代化图形界面、多语言支持、主题切换以及国际象棋 AI 对战功能。

---

## 功能特点

### 国际象棋

- 完整棋盘系统
- 合法走法检测
- 将军与将死检测
- 和棋检测
- 兵升变
- 王车易位
- 吃过路兵
- 双人模式
- AI 对战模式

### UNO

- 完整 UNO 规则实现
- Wild 与 Draw 4 卡牌
- Skip、Reverse、Draw 2 功能
- 多个 AI 玩家
- AI 自动选择颜色
- 胜利判定

### 用户界面

- 浅色主题
- 深色主题
- 英文支持
- 波斯语支持
- 中文支持
- 基于 PyQt6 的现代桌面界面
